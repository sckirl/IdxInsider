# IDX OpenInsider UI/UX Specification

## Design Philosophy
- **Modern & Professional:** Financial terminal aesthetic (Bloomberg/OpenInsider).
- **Dark Mode by Default:** High contrast, data-centric design.
- **Fast & Interactive:** Immediate filtering and sorting.

## Key UI Elements
- **Sticky Header:** Quick search bar and main navigation.
- **Transaction Table:** Sortable, color-coded (Green for BUY, Red for SELL).
- **Filters:** Ticker, Role (Direksi/Komisaris), Date Range, Min Transaction Value.
- **Hover Tooltips:** Show raw disclosure link and ownership percentage details.

## Color Palette
- Background: #0D1117 (GitHub Dark)
- Primary Green: #22C55E (BUY)
- Primary Red: #EF4444 (SELL)
- Text: #E6EDF3 (High legibility)
- Borders: #30363D
